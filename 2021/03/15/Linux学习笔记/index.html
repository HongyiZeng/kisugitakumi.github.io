Linux学习笔记

4 Linux常用命令

4.1 文件处理命令

4.1.1 命令格式

命令格式

命令格式：命令 [-选项] [参数]

例如：

说明：

  • 个别命令使用不遵守此格式；
  • 当有多个选项时可以写在一起；
  • 简化选项和完整选项：-a和--all等价

ls

  • ls：list
  • 命令所在路径：/bin/list
  • 功能：显示目录文件
  • 主要选项：-a：显示所有文件 -l：显示详细信息 -d：查看目录属性
  • 参数：指定要显示的目录

文件权限

  • 第一位：文件类型：- 普通二进制文件； d 目录； l软链接
  • 后九位：每三位一组，每一组对应不同角色的权限。

1.角色类型：u 所有者；g所属组 ；o其他人

2.权限：r读； w写； x执行； -不具有权限

4.1.2 目录处理命令

mkdir

  • mkdir：make directory
  • 命令所在路径：/bin/mkdir
  • 语法：mkdir -p [目录名]
  • 功能描述：创建新目录；选项-p为递归创建，用于多重新目录的创建

cd

  • cd：change directory
  • 命令所在路径：shell内置指令
  • 语法：cd [目录]
  • 功能：切换至对应目录

pwd

  • pwd：print working directory
  • 路径：/bin/pwd
  • 功能：显示当前目录

rmdir

  • rmdir：remove empty directory
  • 路径：/bin/rmdir
  • 语法：rmdir [目录名]
  • 功能：删除==空==目录，很少用

cp

  • cp：copy
  • 路径：/bin/cp
  • 语法：cp [原文件或目录] [目标目录]
  • 选项：-r：复制目录； -p：保留文件属性
  • 功能：复制文件或目录

mv

  • mv：move
  • 路径：/bin/mv
  • 语法：mv [原文件或目录] [目标目录]
  • 功能：剪切文件，改名
  • 补充：ctrl+L或clear命令可以清屏

rm 这个指令慎用

  • rm：remove
  • 路径：/bin/rm
  • 语法：rm [目录或文件]
  • 选项：-r删除目录 -f强制执行
  • 功能：本意是删除文件，但也可以删除目录，要加上选项-r

4.1.3 文件处理命令

touch 创建空文件

  • 语法：touch [文件名]

注意：不建议文件名中使用空格符号，特殊符号，尽管可以使用。

cat 浏览文件内容

  • 语法：cat [文件名]
  • 选项：-n显示行号

  • 反着写命令成tac，可以反向查看文件内容

more 分页显示文件内容

  • 动作：在浏览时，按回车可一行一行查看；按空格或f可一页一页查看；按q可退出浏览。

less 分页显示文件内容，可向上翻页

  • 动作：按pageup一页一页向上翻；按向上箭头一行一行向上翻；输入/+想要搜索的内容并按回车，可以查找内容，接着按n可依次跳转到查找内容的位置。

head 显示文件的前几行

  • 选项：-n 指定行数，不加选项默认为10行

tail 显示文件的末尾几行

  • 额外选项：-f 动态显示文件末尾的内容；即在浏览文件末尾时，文件内容同时发生变化了，会相应地动态显示变化的内容。例如查看日志文件时可以使用该选项。

4.1.4 链接命令

ln 生成链接文件

  • 语法：ln -s [原文件] [目标文件]；不加-s选项则会创建硬链接
  • 选项：-s 创建软链接

软链接特征：

  1. 类似于Windows快捷方式，软链接文件的权限为lrwxrwxrwx，但是最终的权限由原文件来决定。
  2. 软链接文件大小很小，它只是一个符号链接。
  3. 箭头指向原文件。

硬链接特征：

  1. 硬链接实现的功能是：拷贝cp -p + ==同步更新==
  2. 通过i结点识别
  3. 不能跨分区
  4. 不能对目录建立硬链接

4.2 权限管理命令

4.2.1 chmod命令

  • 英文原意：change the permissions mode of a file

  • 语法：chmod [{ugoa}{+-=}{rwx}] [文件或目录]

    ​ chmod [421] [文件或目录]

  • 选项：-R 递归修改

  • 功能：改变目录或文件权限

权限的数字表示

  • r----4 w----2 x----1
  • 例如rwxrw-r--的数字表示形式为764

文件和目录的权限总结

例如：有一个文件夹/temp，权限为777，里面有一个文件test，权限为744。普通用户user对文件test没有写和执行的权限，但仍可以删除掉该文件，因为user对test所在目录temp具有写的权限。

对于目录权限，==有r必定就有x==

4.2.2 其他权限管理命令

chown 改变文件或目录的所有者

  • 英文原意：change file ownership
  • 语法：chown [用户] [文件或目录]

chgrp 改变文件或目录的所属组

  • 语法：chgrp [用户组] [文件或目录]

umask 显示，设置文件的缺省权限

  • 英文原意：the user file-creation mask
  • shell内置指令
  • 语法：umask
  • 选项：-S 以rwx形式显示新建文件的缺省权限
